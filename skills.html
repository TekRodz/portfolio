Compétences

Html / css / scss

Pour faire du développement de site internet, on doit bien passer par le classique html/css. Au niveau du css, j'ai appris à faire de sites responsives et à structurer les pages grâce aux grids et aux flexs. Je sais également utiliser le Scss qui permet une écriture du css mieux organisé, plus lisible et moins répétitive.

Javascript / Jquery

Le Javascript est un langage dont j'ai pu avoir l'utilité lors de mes projets pour faire des dark mods, des menus burger ou encore d'afficher des erreurs. J'ai également appris l'utilisation de la librairie Jquery qui permet de simplifier le Javascript.

Php / Sql

J'ai appris à utiliser le php couplé au sql. Il m'a permis de traiter dans mes projets mes bases de données en créant un système de login / connexion, en affichant des commentaires stocker dans une base ou d'en publier. Le php fait parti de mes langages sur lequel je suis le plus à l'aise.

Three.js

Le three.js est une librairie dont je suis actuellement en apprentissage pour le projet iiMap. Cela me permet d'insérer des éléments 3D dans mon site.

React Native

Le React Native est un framework ayant pour objectif de développer des applications cross-platform donc à la fois compatible pour IOS et Android. J'ai eu l'occasion d'apprendre à m'en servir lors de mon projet Pepif dont l'objectif était de créer une application permettant aux patrouilleurs de centralisé leurs rapports de patrouille.
